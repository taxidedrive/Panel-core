개발자 메모

1) 브라우저에서 직접 Yahoo Finance 비공식 엔드포인트를 치는 방식은 CORS/쿠키 문제로 불안정할 수 있으니 서버 프록시를 두는 편이 안전합니다.
2) 서버는 티커별 실시간 가격만 내려주고, 프리미엄/평단/계약 정보는 이 페이지의 JSON 또는 DB에서 합산하면 됩니다.
3) 새로고침 주기는 30~60초 정도면 충분합니다. 시장 종료 후에는 애프터마켓 값을 별도 필드로 보여줄 수 있습니다.
